# 🐻 팀원 소개
* 정이현 --- Object Settings
* 박인효 --- Player Settings
* 권유리 --- Camera, UI
* 최재원 --- GameManager, SoundManager

# ⚙ 개발 환경

* ``C#``
* ``Unity 2022.3.17f (LTS)``
* ``Visual Studio 2022``
* ``.Net 8.0``
* ``CC0 에셋 사용``
