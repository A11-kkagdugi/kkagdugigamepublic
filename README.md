# 🌲 기능 세부 🌲

### 이동 및 점프
``AD`` 키로 이동합니다.

<br/>

### 장애물
플레이어가 장애물에 부딪히면 게임이 끝납니다.

<br/>

### 점수 시스템
장애물에 부딪히지 않고 버틴 시간이 점수로 나타납니다.
